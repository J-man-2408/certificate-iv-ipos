# Session 4 - Testing:

- the unittest framework is built into python but is still imported. 
- Concepts used in unit testing are:
    - **Test fixture:** The prep needed to run a test and the cleanup
    - **Test case:** The function using TestCase
    - **Test suite:** Groups all the tests together
    - **Test runner:** Runs the tests and conveys results
- Project structure is important. `src` directory and `test` directory
